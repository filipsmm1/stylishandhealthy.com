# Stylishandhealthy fixed responsive site

This folder contains complete replacement source files.

## Deployment structure

- Root: all `.html` files, `robots.txt`, `sitemap.xml`, and `CNAME`
- `/assets/css/styles.css`
- `/assets/js/shared.js`

Keep your existing `/assets/images/` and `/assets/video/` folders on the website. They were not part of the uploaded source package and therefore are not duplicated here.

## Main fixes

- Responsive layouts for small phones, phones, tablets, landscape screens, desktops, and ultrawide displays
- Accessible, scrollable mobile navigation with Escape handling and focus trapping
- 44 px minimum primary touch controls
- iOS-safe 16 px mobile form controls
- Homepage social-grid correction
- Journal CTA/form correction
- Better blog/shop breakpoints
- Safe-area support for notched devices
- Reduced-motion support
- Keyboard focus styles and skip links
- Semantic main landmarks and ARIA status regions
- Lazy loading/async image decoding where safe
- Full replacement files, not partial patches
